Implemented:

- `fp4_mac` now decodes `mac_in.scale` as FP8 E4M3 Ă—256 and applies it as `(fp4 * fp8_activation * scale) >>> 8`
- `fp4_scale_reader`: group-wise scale lookup with default `GROUP_SIZE=16`
- `fp4_systolic_tile`: small multi-lane wrapper around existing `fp4_mac`, validated with zero and non-zero scale-aware vectors
- `fp4_scaled_tile`: connects `fp4_scale_reader` to `fp4_systolic_tile` with 1-cycle valid alignment
- `fp4_systolic_array`: streams K dimension in LANES-wide beats and emits done/result after drain
- `fp4_linear_engine`: minimal multi-output-channel linear engine with preload memories for weights/activations
- `silu_q12_lut`: signed Q12 piecewise-linear SiLU approximation over [-8, +8]
- `rms_norm`: signed Q12 RMSNorm with Newton-Raphson integer sqrt, valid/ready pipeline
- `router_topk`: Q12 dot-product router + Top-2 selection network (4 experts, 8 hidden)
- `expert_ffn_engine`: small Expert FFN prototype (gate/up fp4 linear + SiLU LUT + Q12 down projection)
- `expert_ffn_engine_fp4_down`: gate/up/down all use `fp4_linear_engine`; mid is encoded back to FP8 for down projection
- `gen_ffn_tb_vectors.py`: emits Icarus-compatible packed golden vectors for FFN RTL tests
- Testbenches for scale reader, scalar MAC, tile, scale-connected tile, array, linear engine, SiLU LUT, and Expert FFN
