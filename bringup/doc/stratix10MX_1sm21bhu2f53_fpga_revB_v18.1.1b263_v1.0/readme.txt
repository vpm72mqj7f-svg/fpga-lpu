This preliminary package does not include any hardware design files except schematic.

This readme.txt file accompanies the Stratix 10 MX FPGA Development Kit.

Device target:	1SM21BHU2F53E1VG (Production Device)
ACDS Release:	v18.1.1 build 263
Board revision:	Rev A

//=============================
//    System Requirements
//=============================

Software Requirements
========================
Demo applications installed with the Development Kit require the installation 
of Quartus Prime v18.1.1 and are supported on the following operating systems:
*  Windows 7 (64 and 32 bit)
*  Linux OS (64 and 32 bit)

Hardware Requirements
========================
See the Altera Software Installation and Licensing manual for hardware 
requirements and recommendations for the Quartus Prime software and related
packages.

Demo applications installed with the Development Kit require the following
hardware:
   -  USB port for USB-Blaster(TM) or On-Board USB Blaster(TM) to allow for 
      device programming and debug as well as operating the Board Test System.
